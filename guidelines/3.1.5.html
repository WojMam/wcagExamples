3.1.5 Poziom trudności czytania (AAA)

Gdy tekst wymaga umiejętności czytania bardziej zaawansowanych niż poziom edukacji średniej, dostępna jest dodatkowa treść lub wersja, która nie wymaga umiejętności czytania bardziej zaawansowanych niż poziom edukacji średniej.

Opis kryterium

To kryterium wymaga, aby treści o wysokim poziomie trudności miały alternatywy dostępne dla osób z ograniczonymi umiejętnościami czytania. Tekst nie powinien wymagać umiejętności czytania powyżej poziomu edukacji średniej, a jeśli tak, to musi być dostępna uproszczona wersja lub dodatkowe wyjaśnienia.

Kto odnosi korzyść

  • Osoby z ograniczeniami poznawczymi - łatwiejsze zrozumienie uproszczonych tekstów
  • Osoby z dysleksją - trudności z czytaniem skomplikowanych tekstów
  • Osoby z niepełnosprawnością intelektualną - potrzeba prostego języka
  • Osoby z ADHD - problemy z koncentracją na długich, złożonych tekstach
  • Osoby starsze - możliwe trudności z przetwarzaniem złożonych informacji
  • Osoby uczące się języka - ograniczone słownictwo i znajomość gramatyki
  • Osoby z ograniczoną edukacją - brak doświadczenia z akademickim językiem
  • Młodzi czytelnicy - rozwijające się umiejętności czytania

Wymagania

  • Tekst nie powinien przekraczać poziomu edukacji średniej (po usunięciu nazw własnych i tytułów)
  • Jeśli tekst jest trudniejszy, musi być dostępna alternatywa
  • Alternatywa może być w formie uproszczonej wersji tekstu
  • Alternatywa może być w formie dodatkowych wyjaśnień lub podsumowań
  • Alternatywa musi być łatwo dostępna i wyraźnie oznaczona
  • Kryterium nie dotyczy nazw własnych ani specjalistycznych tytułów

Definicje

Poziom edukacji średniej
Poziom umiejętności czytania odpowiadający ukończeniu szkoły średniej (około 12-13 lat edukacji).
Dodatkowa treść
Uzupełniające materiały, które wyjaśniają lub upraszczają główną treść.
Wersja alternatywna
Równoważna treść napisana prostszym językiem, dostępna jako osobna wersja.
Indeks czytelności
Miara trudności tekstu oparta na długości słów, zdań i złożoności składni.
Prosty język
Styl pisania charakteryzujący się krótkimi zdaniami, powszechnym słownictwem i jasną strukturą.
Nazwy własne
Nazwy osób, miejsc, organizacji i inne specyficzne określenia, które są wyłączone z oceny czytelności.

Poziomy trudności czytania

📚 Poziom podstawowy (klasy 1-6)

  • Bardzo krótkie zdania (5-10 słów)
  • Podstawowe słownictwo
  • Prosta struktura składniowa
  • Konkretne pojęcia

🎓 Poziom gimnazjalny (klasy 7-9)

  • Krótkie do średnich zdania (10-15 słów)
  • Rozszerzone słownictwo
  • Wprowadzenie pojęć abstrakcyjnych
  • Bardziej złożona składnia

🏫 Poziom średni (liceum) - CEL WCAG

  • Średnie zdania (15-20 słów)
  • Zaawansowane słownictwo
  • Pojęcia abstrakcyjne
  • Złożone struktury składniowe

🎓 Poziom wyższy (studia i powyżej)

  • Długie zdania (20+ słów)
  • Specjalistyczne słownictwo
  • Bardzo abstrakcyjne pojęcia
  • Skomplikowane struktury
  • WYMAGA ALTERNATYWY

Przykłady

❌ Nieprawidłowe implementacje

Tekst zbyt trudny bez alternatywy

<article>
    <h2>Implementacja algorytmów kryptograficznych w systemach rozproszonych</h2>
    <p>Zastosowanie asymetrycznych algorytmów kryptograficznych w architekturach 
    rozproszonych wymaga uwzględnienia złożoności obliczeniowej, implikacji 
    bezpieczeństwa oraz efektywności implementacyjnej w kontekście 
    heterogenicznych środowisk wykonawczych.</p>
</article>

Problem: Bardzo wysoki poziom trudności bez żadnej alternatywy.

Alternatywa ukryta lub niedostępna

<article>
    <h2>Raport finansowy</h2>
    <p>Analiza wskaźników rentowności, płynności i zadłużenia...</p>
    <!-- Uproszczona wersja w ukrytym komentarzu -->
    <!-- Nasz zysk wzrósł o 15% w tym roku -->
</article>

Problem: Alternatywa nie jest dostępna dla użytkowników.

✅ Prawidłowe implementacje

Tekst na odpowiednim poziomie

<article>
    <h2>Jak chronić hasła w internecie</h2>

// Funkcja analizy czytelności dla polskiego
class PolishReadabilityAnalyzer {
    constructor() {
        // Częste polskie słowa (nie wymagają uproszczenia)
        this.commonWords = new Set([
            'i', 'w', 'na', 'z', 'do', 'nie', 'jest', 'się', 'że', 'to',
            'od', 'za', 'po', 'być', 'lub', 'jak', 'też', 'już', 'czy'
        ]);
    }
    
    analyze(text) {
        const sentences = this.getSentences(text);
        const words = this.getWords(text);
        const syllables = this.countTotalSyllables(words);
        
        return {
            sentences: sentences.length,
            words: words.length,
            syllables: syllables,
            avgWordsPerSentence: words.length / sentences.length,
            avgSyllablesPerWord: syllables / words.length,
            complexWords: this.getComplexWords(words),
            readabilityScore: this.calculateReadabilityScore(sentences.length, words.length, syllables)
        };
    }
    
    getSentences(text) {
        return text.split(/[.!?]+/).filter(s => s.trim().length > 0);
    }
    
    getWords(text) {
        return text.toLowerCase()
                   .replace(/[^\w\sąęćłńóśźż]/g, ' ')
                   .split(/\s+/)
                   .filter(w => w.length > 0);
    }
    
    countSyllables(word) {
        const vowels = 'aeiouyąęó';
        let count = 0;
        let prevWasVowel = false;
        
        for (let char of word.toLowerCase()) {
            const isVowel = vowels.includes(char);
            if (isVowel && !prevWasVowel) {
                count++;
            }
            prevWasVowel = isVowel;
        }
        
        return Math.max(1, count);
    }
    
    countTotalSyllables(words) {
        return words.reduce((total, word) => total + this.countSyllables(word), 0);
    }
    
    getComplexWords(words) {
        return words.filter(word => 
            word.length > 6 && 
            this.countSyllables(word) > 2 && 
            !this.commonWords.has(word)
        );
    }
    
    calculateReadabilityScore(sentences, words, syllables) {
        // Adaptacja wzoru Flesch dla polskiego
        const avgWordsPerSentence = words / sentences;
        const avgSyllablesPerWord = syllables / words;
        
        return 206.835 - (1.015 * avgWordsPerSentence) - (84.6 * avgSyllablesPerWord);
    }
    
    getReadabilityLevel(score) {
        if (score >= 90) return { level: 'Bardzo łatwy', wcag: true };
        if (score >= 80) return { level: 'Łatwy', wcag: true };
        if (score >= 70) return { level: 'Dość łatwy', wcag: true };
        if (score >= 60) return { level: 'Standardowy', wcag: true };
        if (score >= 50) return { level: 'Dość trudny', wcag: true };
        if (score >= 30) return { level: 'Trudny', wcag: false };
        return { level: 'Bardzo trudny', wcag: false };
    }
    
    getSuggestions(analysis) {
        const suggestions = [];
        
        if (analysis.avgWordsPerSentence > 20) {
            suggestions.push({
                type: 'sentence_length',
                message: 'Skróć zdania - średnio ' + analysis.avgWordsPerSentence.toFixed(1) + ' słów na zdanie',
                priority: 'high'
            });
        }
        
        if (analysis.complexWords.length > analysis.words.length * 0.1) {
            suggestions.push({
                type: 'vocabulary',
                message: 'Zmniejsz liczbę skomplikowanych słów (' + analysis.complexWords.length + ' z ' + analysis.words.length + ')',
                priority: 'medium'
            });
        }
        
        if (analysis.avgSyllablesPerWord > 2.5) {
            suggestions.push({
                type: 'word_complexity',
                message: 'Używaj prostszych słów - średnio ' + analysis.avgSyllablesPerWord.toFixed(1) + ' sylab na słowo',
                priority: 'medium'
            });
        }
        
        return suggestions;
    }
}

// Użycie
const analyzer = new PolishReadabilityAnalyzer();
const text = document.body.textContent;
const analysis = analyzer.analyze(text);
const level = analyzer.getReadabilityLevel(analysis.readabilityScore);
const suggestions = analyzer.getSuggestions(analysis);

    poziom: level.level,
    zgodneZWCAG: level.wcag,
    sugestie: suggestions
});

// Plugin sprawdzający czytelność postów
class ReadabilityChecker {
    
    public function __construct() {
        add_action('post_submitbox_misc_actions', array($this, 'add_readability_check'));
        add_action('save_post', array($this, 'save_readability_score'));
        add_action('wp_enqueue_scripts', array($this, 'enqueue_scripts'));
    }
    
    public function add_readability_check() {
        global $post;
        $score = get_post_meta($post->ID, '_readability_score', true);
        $level = $this->get_readability_level($score);
        
        echo '<div class="misc-pub-section readability-check">';
        echo '<span>Czytelność: <strong class="readability-' . $level['class'] . '">' . $level['label'] . '</strong></span>';
        if (!$level['wcag_compliant']) {
            echo '<br><small style="color: #d63638;">⚠️ Może wymagać uproszczonej wersji</small>';
        }
        echo '</div>';
    }
    
    public function save_readability_score($post_id) {
        if (defined('DOING_AUTOSAVE') && DOING_AUTOSAVE) return;
        
        $content = get_post_field('post_content', $post_id);
        $score = $this->calculate_readability($content);
        
        update_post_meta($post_id, '_readability_score', $score);
        
        // Automatyczne dodanie pola dla uproszczonej wersji
        if ($score < 50) {
            $simplified = get_post_meta($post_id, '_simplified_version', true);
            if (empty($simplified)) {
                update_post_meta($post_id, '_needs_simplified_version', true);
            }
        }
    }
    
    public function calculate_readability($text) {
        $text = strip_tags($text);
        $sentences = preg_split('/[.!?]+/', $text);
        $sentences = array_filter($sentences, function($s) { return trim($s) !== ''; });
        
        $words = str_word_count($text, 1, 'ąęćłńóśźż');
        $syllables = 0;
        
        foreach ($words as $word) {
            $syllables += $this->count_syllables($word);
        }
        
        if (count($sentences) == 0 || count($words) == 0) return 0;
        
        $avg_words_per_sentence = count($words) / count($sentences);
        $avg_syllables_per_word = $syllables / count($words);
        
        return 206.835 - (1.015 * $avg_words_per_sentence) - (84.6 * $avg_syllables_per_word);
    }
    
    private function count_syllables($word) {
        $word = strtolower($word);
        $vowels = 'aeiouyąęó';
        $count = 0;
        $prev_was_vowel = false;
        
        for ($i = 0; $i < strlen($word); $i++) {
            $is_vowel = strpos($vowels, $word[$i]) !== false;
            if ($is_vowel && !$prev_was_vowel) {
                $count++;
            }
            $prev_was_vowel = $is_vowel;
        }
        
        return max(1, $count);
    }
    
    private function get_readability_level($score) {
        if ($score >= 50) {
            return array('label' => 'Odpowiednia', 'class' => 'good', 'wcag_compliant' => true);
        } elseif ($score >= 30) {
            return array('label' => 'Trudna', 'class' => 'warning', 'wcag_compliant' => false);
        } else {
            return array('label' => 'Bardzo trudna', 'class' => 'error', 'wcag_compliant' => false);
        }
    }
}

                    <p>Ta treść została napisana prostszym językiem dla lepszego zrozumienia.</p>
                </div>
            )}
        </div>
    );
};

Automatyczne generowanie uproszczonych wersji

class TextSimplifier {
    constructor() {
        // Słownik synonimów prostych
        this.synonyms = new Map([
            ['wykorzystać', 'użyć'],
            ['implementować', 'wdrożyć'],
            ['zrealizować', 'wykonać'],
            ['specyfikacja', 'opis'],
            ['funkcjonalność', 'funkcja'],
            ['procedura', 'sposób'],
            ['metodologia', 'metoda'],
            ['dokumentacja', 'opis'],
            ['administracja', 'zarządzanie'],
            ['optymalizacja', 'poprawa']
        ]);
        
        // Wzorce do upraszczania
        this.simplificationRules = [
            {
                pattern: /w celu (.+)/gi,
                replacement: 'żeby $1'
            },
            {
                pattern: /jest możliwe (.+)/gi,
                replacement: 'można $1'
            },
            {
                pattern: /należy (.+)/gi,
                replacement: 'musisz $1'
            },
            {
                pattern: /w przypadku gdy (.+)/gi,
                replacement: 'jeśli $1'
            }
        ];
    }
    
    simplifyText(text) {
        let simplified = text;
        
        // Zastąp skomplikowane słowa prostszymi
        for (let [complex, simple] of this.synonyms) {
            const regex = new RegExp('\\b' + complex + '\\b', 'gi');
            simplified = simplified.replace(regex, simple);
        }
        
        // Zastosuj wzorce upraszczania
        for (let rule of this.simplificationRules) {
            simplified = simplified.replace(rule.pattern, rule.replacement);
        }
        
        // Podziel długie zdania
        simplified = this.splitLongSentences(simplified);
        
        // Zmień bierną stronę na czynną
        simplified = this.convertPassiveToActive(simplified);
        
        return simplified;
    }
    
    splitLongSentences(text) {
        const sentences = text.split(/(?<=[.!?])\s+/);
        const simplified = [];
        
        for (let sentence of sentences) {
            const words = sentence.split(/\s+/);
            if (words.length > 20) {
                // Znajdź punkt podziału (spójniki)
                const conjunctions = ['i', 'ale', 'oraz', 'albo', 'jednak', 'ponieważ', 'dlatego'];
                let splitPoint = -1;
                
                for (let i = 10; i < words.length - 5; i++) {
                    if (conjunctions.includes(words[i].toLowerCase())) {
                        splitPoint = i;
                        break;
                    }
                }
                
                if (splitPoint > -1) {
                    const firstPart = words.slice(0, splitPoint).join(' ') + '.';
                    const secondPart = words.slice(splitPoint + 1).join(' ');
                    simplified.push(firstPart);
                    simplified.push(secondPart);
                } else {
                    simplified.push(sentence);
                }
            } else {
                simplified.push(sentence);
            }
        }
        
        return simplified.join(' ');
    }
    
    convertPassiveToActive(text) {
        // Podstawowe wzorce strony biernej w polskim
        const passivePatterns = [
            {
                pattern: /(.+) został (.+) przez (.+)/gi,
                replacement: '$3 $2 $1'
            },
            {
                pattern: /(.+) zostanie (.+)/gi,
                replacement: 'ktoś $2 $1'
            }
        ];
        
        let converted = text;
        for (let pattern of passivePatterns) {
            converted = converted.replace(pattern.pattern, pattern.replacement);
        }
        
        return converted;
    }
    
    generateSimplifiedVersion(originalText, targetLevel = 50) {
        let simplified = this.simplifyText(originalText);
        let attempts = 0;
        const maxAttempts = 3;
        
        while (attempts < maxAttempts) {
            const score = this.calculateReadabilityScore(simplified);
            if (score >= targetLevel) break;
            
            // Dodatkowe uproszczenia
            simplified = this.furtherSimplify(simplified);
            attempts++;
        }
        
        return {
            text: simplified,
            score: this.calculateReadabilityScore(simplified),
            improvements: this.getImprovements(originalText, simplified)
        };
    }
    
    furtherSimplify(text) {
        // Zamień przecinki na kropki gdzie to możliwe
        let simplified = text.replace(/([^,]{20,}), ([^,]{10,})/g, '$1. $2');
        
        // Usuń niepotrzebne słowa
        const fillerWords = ['oczywiście', 'jednakże', 'w rzeczywistości', 'rzeczywiście'];
        for (let word of fillerWords) {
            const regex = new RegExp('\\b' + word + '\\b,?\\s*', 'gi');
            simplified = simplified.replace(regex, '');
        }
        
        return simplified;
    }
    
    calculateReadabilityScore(text) {
        // Uproszczona implementacja Flesch
        const sentences = text.split(/[.!?]+/).filter(s => s.trim());
        const words = text.split(/\s+/).filter(w => w.trim());
        
        if (sentences.length === 0 || words.length === 0) return 0;
        
        const avgWordsPerSentence = words.length / sentences.length;
        const avgSyllablesPerWord = words.reduce((total, word) => 
            total + this.countSyllables(word), 0) / words.length;
        
        return 206.835 - (1.015 * avgWordsPerSentence) - (84.6 * avgSyllablesPerWord);
    }
    
    countSyllables(word) {
        const vowels = 'aeiouyąęó';
        let count = 0;
        let prevWasVowel = false;
        
        for (let char of word.toLowerCase()) {
            const isVowel = vowels.includes(char);
            if (isVowel && !prevWasVowel) count++;
            prevWasVowel = isVowel;
        }
        
        return Math.max(1, count);
    }
    
    getImprovements(original, simplified) {
        const originalScore = this.calculateReadabilityScore(original);
        const simplifiedScore = this.calculateReadabilityScore(simplified);
        
        return {
            scoreImprovement: simplifiedScore - originalScore,
            lengthReduction: ((original.length - simplified.length) / original.length * 100).toFixed(1) + '%',
            readabilityGain: simplifiedScore >= 50 ? 'Osiągnięto cel WCAG' : 'Wymaga dalszej pracy'
        };
    }
}

const result = simplifier.generateSimplifiedVersion(originalText);

console.log('Oryginalny:', originalText);
console.log('Uproszczony:', result.text);
console.log('Poprawa wyniku:', result.improvements);

class MultilingualReadabilityManager {
    constructor() {
        this.languages = {
            'pl': {
                name: 'Polski',
                vowels: 'aeiouyąęó',
                avgSyllablesPerWord: 2.1,
                avgWordsPerSentence: 16,
                complexWordThreshold: 3
            },
            'en': {
                name: 'English',
                vowels: 'aeiouy',
                avgSyllablesPerWord: 1.5,
                avgWordsPerSentence: 15,
                complexWordThreshold: 3
            }
        };
        
        this.readabilityFormulas = {
            'flesch': this.calculateFlesch,
            'kincaid': this.calculateKincaid,
            'gunning': this.calculateGunningFog
        };
    }
    
    analyzeMultilingualContent(content, primaryLanguage = 'pl') {
        const sections = this.detectLanguageSections(content);
        const results = {};
        
        for (let [lang, text] of sections) {
            results[lang] = this.analyzeLanguageSection(text, lang);
        }
        
        return {
            primary: primaryLanguage,
            sections: results,
            overallCompliance: this.checkOverallCompliance(results),
            recommendations: this.getMultilingualRecommendations(results)
        };
    }
    
    detectLanguageSections(content) {
        // Uproszczone wykrywanie sekcji językowych po atrybucie lang
        const sections = new Map();
        const parser = new DOMParser();
        const doc = parser.parseFromString(content, 'text/html');
        
        // Główny język
        const mainLang = doc.documentElement.lang || 'pl';
        let mainText = doc.body.textContent;
        
        // Znajdź elementy z innym językiem
        const foreignElements = doc.querySelectorAll('[lang]:not([lang="' + mainLang + '"])');
        
        foreignElements.forEach(element => {
            const lang = element.getAttribute('lang');
            const text = element.textContent;
            
            if (!sections.has(lang)) {
                sections.set(lang, '');
            }
            sections.set(lang, sections.get(lang) + ' ' + text);
            
            // Usuń z tekstu głównego
            mainText = mainText.replace(text, '');
        });
        
        sections.set(mainLang, mainText);
        
        return sections;
    }
    
    analyzeLanguageSection(text, language) {
        const langConfig = this.languages[language] || this.languages['pl'];
        const analysis = this.basicTextAnalysis(text, langConfig);
        
        return {
            language: langConfig.name,
            wordCount: analysis.words,
            sentenceCount: analysis.sentences,
            avgWordsPerSentence: analysis.avgWordsPerSentence,
            readabilityScores: {
                flesch: this.calculateFlesch(analysis, langConfig),
                kincaid: this.calculateKincaid(analysis, langConfig),
                gunning: this.calculateGunningFog(analysis, langConfig)
            },
            wcagCompliant: this.checkWCAGCompliance(analysis, langConfig),
            suggestions: this.getLanguageSpecificSuggestions(analysis, langConfig)
        };
    }
    
    basicTextAnalysis(text, langConfig) {
        const sentences = text.split(/[.!?]+/).filter(s => s.trim().length > 0);
        const words = text.split(/\s+/).filter(w => w.trim().length > 0);
        const syllables = words.reduce((total, word) => 
            total + this.countSyllables(word, langConfig.vowels), 0);
        
        return {
            sentences: sentences.length,
            words: words.length,
            syllables: syllables,
            avgWordsPerSentence: words.length / sentences.length,
            avgSyllablesPerWord: syllables / words.length,
            complexWords: this.getComplexWords(words, langConfig)
        };
    }
    
    countSyllables(word, vowels) {
        let count = 0;
        let prevWasVowel = false;
        
        for (let char of word.toLowerCase()) {
            const isVowel = vowels.includes(char);
            if (isVowel && !prevWasVowel) count++;
            prevWasVowel = isVowel;
        }
        
        return Math.max(1, count);
    }
    
    getComplexWords(words, langConfig) {
        return words.filter(word => 
            this.countSyllables(word, langConfig.vowels) >= langConfig.complexWordThreshold
        );
    }
    
    calculateFlesch(analysis, langConfig) {
        return 206.835 - (1.015 * analysis.avgWordsPerSentence) - (84.6 * analysis.avgSyllablesPerWord);
    }
    
    calculateKincaid(analysis, langConfig) {
        return (0.39 * analysis.avgWordsPerSentence) + (11.8 * analysis.avgSyllablesPerWord) - 15.59;
    }
    
    calculateGunningFog(analysis, langConfig) {
        const complexWordPercentage = (analysis.complexWords.length / analysis.words) * 100;
        return 0.4 * (analysis.avgWordsPerSentence + complexWordPercentage);
    }
    
    checkWCAGCompliance(analysis, langConfig) {
        const fleschScore = this.calculateFlesch(analysis, langConfig);
        const gradeLevel = this.calculateKincaid(analysis, langConfig);
        
        return {
            flesch: fleschScore >= 50,
            grade: gradeLevel <= 12,
            overall: fleschScore >= 50 && gradeLevel <= 12
        };
    }
    
    checkOverallCompliance(results) {
        let compliantSections = 0;
        let totalSections = 0;
        
        for (let [lang, analysis] of Object.entries(results)) {
            totalSections++;
            if (analysis.wcagCompliant.overall) {
                compliantSections++;
            }
        }
        
        return {
            compliantSections: compliantSections,
            totalSections: totalSections,
            percentage: (compliantSections / totalSections * 100).toFixed(1),
            isCompliant: compliantSections === totalSections
        };
    }
    
    getMultilingualRecommendations(results) {
        const recommendations = [];
        
        for (let [lang, analysis] of Object.entries(results)) {
            if (!analysis.wcagCompliant.overall) {
                recommendations.push({
                    language: analysis.language,
                    priority: 'high',
                    message: `Sekcja w języku ${analysis.language} wymaga uproszczenia lub alternatywy`,
                    suggestions: analysis.suggestions
                });
            }
        }
        
        return recommendations;
    }
    
    getLanguageSpecificSuggestions(analysis, langConfig) {
        const suggestions = [];
        
        if (analysis.avgWordsPerSentence > langConfig.avgWordsPerSentence + 5) {
            suggestions.push('Skróć zdania - są za długie dla tego języka');
        }
        
        if (analysis.avgSyllablesPerWord > langConfig.avgSyllablesPerWord + 0.5) {
            suggestions.push('Używaj prostszych słów o mniejszej liczbie sylab');
        }
        
        const complexWordPercentage = (analysis.complexWords.length / analysis.words) * 100;
        if (complexWordPercentage > 15) {
            suggestions.push('Zmniejsz liczbę skomplikowanych słów (' + complexWordPercentage.toFixed(1) + '%)');
        }
        
        return suggestions;
    }
}
