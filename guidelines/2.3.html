2.3 Ataki padaczki i reakcje fizyczne

Wytyczna ta zapewnia, że treść nie powoduje napadów padaczkowych lub innych reakcji fizycznych.

Opis wytycznej

Niektóre osoby z padaczką fotogenną mogą doświadczyć napadów wywołanych przez migające lub błyskające treści. Miganie w określonych częstotliwościach może wywoływać napady padaczkowe nawet u osób, które wcześniej ich nie doświadczyły.

Ta wytyczna zapewnia, że treść nie zawiera elementów, które migają więcej niż trzy razy w ciągu jednej sekundy, lub że miganie znajduje się poniżej ogólnych progów wywołujących napady padaczkowe.

Dodatkowo, niektóre rodzaje ruchów lub wzorców wizualnych mogą wywoływać reakcje vestibuularne (związane z układem równowagi), powodując zawroty głowy, nudności lub dezorientację.

⚠️ Ważne uwagi bezpieczeństwa

Ostrzeżenie dotyczące bezpieczeństwa

  • Nigdy nie testuj migających treści na sobie ani na innych osobach - może to być niebezpieczne
  • Używaj wyłącznie narzędzi automatycznych do analizy migających treści
  • W przypadku wątpliwości, unikaj stosowania migających elementów całkowicie
  • Zawsze umieść ostrzeżenia przed treściami, które mogą zawierać miganie

Typowe wyzwalacze napadów

Elementy mogące powodować napady padaczkowe:

  • Migające obrazy - częstotliwość powyżej 3 Hz (3 błyski na sekundę)
  • Błyskające światła - szczególnie w kolorach czerwonym i niebieskim
  • Szybko zmieniające się kolory - kontrastowe przejścia
  • Stroboskopowe efekty - regularne, szybkie zmiany jasności
  • Szybkie ruchy kamery - w materiałach wideo
  • Geometryczne wzory - regularnie powtarzające się paski lub kratki

Elementy mogące powodować reakcje vestibularne:

  • Animacje paralaksy - przesuwające się warstwy z różną prędkością
  • Zoom i obracanie - szybkie zmiany skali lub orientacji
  • Animacje przejść - intensywne efekty przy zmianie stron
  • Auto-play video - automatycznie odtwarzane materiały z ruchem

Strategie zapobiegania

Najlepsze praktyki:

  • Unikaj migających elementów - jeśli nie są absolutnie konieczne
  • Testuj częstotliwość migania - upewnij się, że jest poniżej 3 Hz
  • Dodaj kontrolę użytkownika - pozwól wyłączyć animacje
  • Przestrzegaj preferencji systemu - obsłuż `prefers-reduced-motion`
  • Umieść ostrzeżenia - informuj o potencjalnie problematycznych treściach
  • Zapewnij alternatywy - oferuj statyczne wersje animowanych treści

Wsparcie CSS dla dostępności ruchu

/* Respektowanie preferencji użytkownika dotyczących ruchu */
@media (prefers-reduced-motion: reduce) {
    * {
        animation-duration: 0.01ms !important;
        animation-iteration-count: 1 !important;
        transition-duration: 0.01ms !important;
        scroll-behavior: auto !important;
    }
}

/* Bezpieczne animacje - wolne i subtelne */
.safe-animation {
    animation: fadeIn 0.5s ease-in-out;
}

@keyframes fadeIn {
    from { opacity: 0; }
    to { opacity: 1; }
}

/* Unikaj szybkich migań */
.avoid-flashing {
    /* Nigdy nie używaj animacji szybszych niż 3 klatki na sekundę */
    animation-duration: 2s; /* minimum dla bezpieczeństwa */
}
