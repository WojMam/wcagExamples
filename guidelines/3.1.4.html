Przejdź do treści głównej

3.1.4 Skróty (AAA)

Mechanizm identyfikacji rozwinięcia lub znaczenia skrótów jest dostępny.

Opis kryterium

Wszystkie skróty (włączając akronimy i inicjalizmy) muszą mieć dostępne rozwinięcie lub wyjaśnienie ich znaczenia. Pozwala to użytkownikom zrozumieć treść zawierającą specjalistyczne lub branżowe skróty, które mogą być niezrozumiałe dla szerszej publiczności.

Kto odnosi korzyść

  • Osoby z ograniczeniami poznawczymi - łatwiejsze zrozumienie skróconych form
  • Osoby z dysleksją - dodatkowe wsparcie w rozumieniu tekstów
  • Osoby uczące się języka - pomoc w rozumieniu lokalnych skrótów
  • Nowi użytkownicy w dziedzinie - wyjaśnienie branżowych skrótów
  • Osoby starsze - objaśnienie nowoczesnych akronimów
  • Czytniki ekranu - prawidłowa wymowa i rozumienie skrótów
  • Tłumacze - lepsze zrozumienie kontekstu dla tłumaczeń

Wymagania

  • Mechanizm wyjaśniania skrótów musi być dostępny
  • Rozwinięcia muszą być jasne i precyzyjne
  • Mechanizm może obejmować różne sposoby dostarczania wyjaśnień
  • Wyjaśnienia muszą być dostępne dla technologii wspomagających
  • Skróty mogą być wyjaśnione przy pierwszym wystąpieniu lub w dedykowanej sekcji

Definicje

Skrót
Skrócona forma słowa lub wyrażenia, utworzona przez pominięcie części liter.
Akronim
Skrót utworzony z pierwszych liter słów, wymawiane jako słowo (np. "NASA", "radar").
Inicjalizm
Skrót utworzony z pierwszych liter słów, wymawiane literka po literce (np. "HTML", "CSS").
Rozwinięcie
Pełna forma skrótu pokazująca, jakie słowa reprezentuje.
Wyjaśnienie znaczenia
Opis tego, co skrót oznacza lub reprezentuje w danym kontekście.

Przykłady

❌ Nieprawidłowe implementacje

Skróty bez wyjaśnień

<p>Nasz system CRM integruje się z API poprzez REST, 
wykorzystując JSON do wymiany danych z bazą SQL.</p>

Problem: Skróty CRM, API, REST, JSON, SQL nie są wyjaśnione.

Niejednoznaczne skróty

<p>Departament HR współpracuje z IT w zakresie AI.</p>

Problem: Skróty mogą mieć różne znaczenia w różnych kontekstach.

Skróty tylko w języku obcym

<p><abbr title="Chief Executive Officer">CEO</abbr> firmy ogłosił nową strategię.</p>

Problem: Rozwinięcie tylko w języku angielskim dla polskiej strony.

✅ Prawidłowe implementacje

Element abbr z rozwinięciem

<p>Nasz system <abbr title="Customer Relationship Management">CRM</abbr> 
integruje się z <abbr title="Application Programming Interface">API</abbr> 
poprzez <abbr title="Representational State Transfer">REST</abbr>.</p>

Dlaczego działa: Każdy skrót ma pełne rozwinięcie w atrybucie title.

Pierwsze wystąpienie z rozwinięciem

<p>World Wide Web Consortium (<abbr>W3C</abbr>) ustala standardy internetowe. 
Wytyczne <abbr>W3C</abbr> są powszechnie akceptowane.</p>

Dlaczego działa: Pierwsze wystąpienie zawiera pełne rozwinięcie, kolejne mogą używać tylko skrótu.

Kontekstowe wyjaśnienia

<p>Firma stosuje metodologię 
<dfn><abbr title="Agile Software Development">Agile</abbr></dfn> 
do zarządzania projektami programistycznymi.</p>

<p>W medycynie <dfn><abbr title="Magnetic Resonance Imaging">MRI</abbr></dfn> 
to zaawansowana technika obrazowania.</p>

Dlaczego działa: Kombinacja dfn i abbr zapewnia semantyczne oznaczenie definicji.

Dwujęzyczne wyjaśnienia

<p><abbr title="Chief Executive Officer - Dyrektor Generalny">CEO</abbr> 
firmy <abbr title="International Business Machines - Międzynarodowe Maszyny Biznesowe">IBM</abbr> 
ogłosił nową strategię.</p>

Dlaczego działa: Rozwinięcie zawiera zarówno oryginalną formę jak i polskie tłumaczenie.

Rodzaje skrótów

📝 Standardowe skróty

  • Skróty słów: dr (doktor), prof. (profesor), ul. (ulica)
  • Skróty jednostek: kg (kilogram), cm (centymetr), h (godzina)
  • Skróty geograficzne: woj. (województwo), pow. (powiat)
  • Skróty czasowe: r. (rok), w. (wiek), dn. (dzień)

🔤 Akronimy

  • Organizacje: NASA, UNESCO, WHO
  • Technologie: AJAX, JSON, YAML
  • Standardy: ASCII, JPEG, MPEG
  • Protokoły: HTTP, SMTP, FTP

📊 Inicjalizmy

  • Techniczne: HTML, CSS, XML, SQL
  • Biznesowe: CEO, CTO, HR, PR
  • Akademickie: PhD, MBA, MSc, BSc
  • Medyczne: MRI, CT, EKG, USG

⚡ Skróty kontekstowe

  • IT: API, SDK, IDE, CLI
  • Marketing: SEO, SEM, CTR, CPC
  • Finanse: ROI, KPI, P&L, EBITDA
  • Prawo: ZUS, NFZ, GUS, UE

// Wykrywanie potencjalnych skrótów
function detectPotentialAbbreviations() {
    const patterns = [
        // Akronimy (2-6 wielkich liter)
        /\b[A-Z]{2,6}\b/g,
        // Skróty z kropkami
        /\b[A-Za-z]+\./g,
        // Powszechne skróty techniczne
        /\b(API|SDK|HTML|CSS|XML|JSON|HTTP|HTTPS|FTP|SSH|SQL|NoSQL|REST|SOAP|AJAX|DOM|URL|URI|IP|TCP|UDP|DNS|SSL|TLS|OAuth|JWT|CRUD|MVC|ORM|IDE|CLI|GUI|OS|CPU|GPU|RAM|SSD|HDD|USB|WiFi|LAN|WAN|VPN)\b/gi,
        // Skróty biznesowe
        /\b(CEO|CTO|CFO|CMO|HR|PR|ROI|KPI|B2B|B2C|CRM|ERP|SaaS|PaaS|IaaS|AI|ML|IoT|AR|VR|UX|UI)\b/gi
    ];
    
    const pageText = document.body.textContent;
    const foundAbbreviations = new Set();
    
    patterns.forEach(pattern => {
        const matches = pageText.match(pattern);
        if (matches) {
            matches.forEach(match => {
                // Odfiltruj oczywiste słowa (np. "IT" w "GitHub")
                if (match.length >= 2 && match !== match.toLowerCase()) {
                    foundAbbreviations.add(match);
                }
            });
        }
    });
    
    return Array.from(foundAbbreviations);
}

// Sprawdzanie czy skróty mają wyjaśnienia
function checkAbbreviationDefinitions() {
    const potentialAbbrs = detectPotentialAbbreviations();
    const undefinedAbbrs = [];
    
    potentialAbbrs.forEach(abbr => {
        // Sprawdź różne sposoby definicji
        const hasDefinition = 
            // Element abbr z title
            document.querySelector(`abbr[title]:contains("${abbr}")`) ||
            // Rozwinięcie w nawiasach
            document.body.textContent.includes(`(${abbr})`) ||
            document.body.textContent.includes(`${abbr} (`) ||
            // Link do słowniczka
            document.querySelector(`a[href*="${abbr.toLowerCase()}"]`) ||
            // Definicja w słowniczku
            document.querySelector(`#glossary [id*="${abbr.toLowerCase()}"], #abbreviations-glossary [id*="${abbr.toLowerCase()}"]`);
            
        if (!hasDefinition) {
            undefinedAbbrs.push(abbr);
        }
    });
    
    return {
        totalAbbreviations: potentialAbbrs.length,
        undefinedAbbreviations: undefinedAbbrs,
        definitionCoverage: potentialAbbrs.length > 0 ? 
            ((potentialAbbrs.length - undefinedAbbrs.length) / potentialAbbrs.length * 100).toFixed(1) : 100
    };
}

// Walidacja elementów abbr
function validateAbbrElements() {
    const issues = [];
    const abbrElements = document.querySelectorAll('abbr');
    
    abbrElements.forEach((abbr, index) => {
        const title = abbr.getAttribute('title');
        const content = abbr.textContent.trim();
        
        // Sprawdź czy ma title lub inne wyjaśnienie
        if (!title) {
            // Sprawdź czy jest w kontekście z rozwinięciem
            const parent = abbr.parentElement;
            const parentText = parent ? parent.textContent : '';
            const hasContextualExpansion = 
                parentText.includes(`(${content})`) ||
                parentText.includes(`${content} (`) ||
                parent.querySelector(`dfn`);
                
            if (!hasContextualExpansion) {
                issues.push(`Element abbr "${content}" nie ma atrybutu title ani kontekstowego wyjaśnienia`);
            }
        }
        
        // Sprawdź czy title nie jest pusty
        if (title && title.trim() === '') {
            issues.push(`Element abbr "${content}" ma pusty atrybut title`);
        }
        
        // Sprawdź czy title różni się od zawartości
        if (title && title.trim().toLowerCase() === content.toLowerCase()) {
            issues.push(`Element abbr "${content}" ma identyczną zawartość i title`);
        }
    });
    
    return issues;
}

// Sprawdzenie dostępności skrótów
function checkAbbreviationAccessibility() {
    const issues = [];
    
    // Sprawdź czy skróty są czytelne dla czytników ekranu
    const abbrWithoutLang = document.querySelectorAll('abbr:not([lang]):not([title])');
    if (abbrWithoutLang.length > 0) {
        issues.push(`${abbrWithoutLang.length} elementów abbr bez title - mogą być źle wymawiane`);
    }
    
    // Sprawdź czy długie akronimy mają spacje dla lepszej wymowy
    const longAcronyms = document.querySelectorAll('abbr');
    longAcronyms.forEach(abbr => {
        const content = abbr.textContent;
        if (content.length > 4 && /^[A-Z]+$/.test(content)) {
            const title = abbr.getAttribute('title');
            if (!title || !title.includes(' ')) {
                issues.push(`Długi akronim "${content}" może być trudny do wymówienia`);
            }
        }
    });
    
    return issues;
}

// Uruchom wszystkie testy
const abbrAnalysis = checkAbbreviationDefinitions();
const abbrValidation = validateAbbrElements();
const accessibilityIssues = checkAbbreviationAccessibility();

console.log('Analiza skrótów:', abbrAnalysis);
console.log('Problemy z elementami abbr:', abbrValidation);
console.log('Problemy z dostępnością:', accessibilityIssues);

// Shortcode dla skrótów
function abbreviation_shortcode($atts, $content = null) {
    $atts = shortcode_atts(array(
        'title' => '',
        'explain' => 'false' // czy pokazać rozwinięcie w tekście
    ), $atts);
    
    if (!$content || !$atts['title']) {
        return $content;
    }
    
    if ($atts['explain'] === 'true') {
        return sprintf(
            '%s (<abbr title="%s">%s</abbr>)',
            esc_html($atts['title']),
            esc_attr($atts['title']),
            esc_html($content)
        );
    }
    
    return sprintf(
        '<abbr title="%s">%s</abbr>',
        esc_attr($atts['title']),
        esc_html($content)
    );
}
add_shortcode('abbr', 'abbreviation_shortcode');

// Funkcja automatycznego wykrywania skrótów
function auto_add_abbreviations($content) {
    $abbreviations = array(
        'HTML' => 'HyperText Markup Language',
        'CSS' => 'Cascading Style Sheets',
        'JS' => 'JavaScript',
        'API' => 'Application Programming Interface',
        'JSON' => 'JavaScript Object Notation',
        'XML' => 'eXtensible Markup Language'
    );
    
    foreach ($abbreviations as $abbr => $expansion) {
        // Dodaj abbr tylko do pierwszego wystąpienia
        $pattern = '/\b' . preg_quote($abbr, '/') . '\b/';
        $content = preg_replace(
            $pattern,
            '<abbr title="' . $expansion . '">' . $abbr . '</abbr>',
            $content,
            1 // tylko pierwsze wystąpienie
        );
    }
    
    return $content;
}
add_filter('the_content', 'auto_add_abbreviations');

import React, { createContext, useContext } from 'react';

// Kontekst dla słowniczka skrótów
const AbbreviationContext = createContext({});

// Provider dla słowniczka
export const AbbreviationProvider = ({ children, abbreviations = {} }) => {
    return (
        <AbbreviationContext.Provider value={abbreviations}>
            {children}
        </AbbreviationContext.Provider>
    );
};

// Komponent skrótu
const Abbreviation = ({ 
    children, 
    title, 
    firstOccurrence = false,
    definition = false
}) => {
    const abbreviations = useContext(AbbreviationContext);
    
    // Automatyczne pobieranie definicji ze słowniczka
    const expansion = title || abbreviations[children] || '';
    
    if (firstOccurrence && expansion) {
        return (
            <span>
                {expansion} (<abbr title={expansion}>{children}</abbr>)
            </span>
        );
    }
    
    if (definition && expansion) {
        return (
            <dfn>
                <abbr title={expansion}>{children}</abbr>
            </dfn>
        );
    }
    
    return (
        <abbr title={expansion}>
            {children}
        </abbr>
    );
};

// Hook do zarządzania skrótami
const useAbbreviation = (abbr) => {
    const abbreviations = useContext(AbbreviationContext);
    return abbreviations[abbr] || '';
};

// AbbreviationPlugin.js
export default {
    install(app, options = {}) {
        // Globalny słowniczek skrótów
        app.config.globalProperties.$abbreviations = options.abbreviations || {};
        
        // Komponent skrótu
        app.component('Abbreviation', {
            props: {
                title: String,
                firstOccurrence: Boolean,
                definition: Boolean
            },
            template: `
                <span v-if="firstOccurrence && expansion">
                    {{ expansion }} (<abbr :title="expansion"><slot></slot></abbr>)
                </span>
                <dfn v-else-if="definition && expansion">
                    <abbr :title="expansion"><slot></slot></abbr>
                </dfn>
                <abbr v-else :title="expansion">
                    <slot></slot>
                </abbr>
            `,
            computed: {
                expansion() {
                    const abbr = this.$slots.default()[0].children;
                    return this.title || this.$abbreviations[abbr] || '';
                }
            }
        });
        
        // Dyrektywa dla automatycznego dodawania skrótów
        app.directive('abbreviations', {
            mounted(el) {
                const abbreviations = app.config.globalProperties.$abbreviations;
                let html = el.innerHTML;
                
                Object.entries(abbreviations).forEach(([abbr, expansion]) => {
                    const regex = new RegExp(`\\b${abbr}\\b`, 'g');
                    html = html.replace(
                        regex,
                        `<abbr title="${expansion}">${abbr}</abbr>`
                    );
                });
                
                el.innerHTML = html;
            }
        });
    }
};

class ContextualAbbreviationManager {
    constructor() {
        this.abbreviations = new Map();
        this.contexts = new Map();
        this.processed = new Set();
    }
    
    // Dodaj definicję z kontekstem
    addAbbreviation(abbr, expansion, context = 'general') {
        if (!this.abbreviations.has(abbr)) {
            this.abbreviations.set(abbr, new Map());
        }
        this.abbreviations.get(abbr).set(context, expansion);
    }
    
    // Dodaj wiele skrótów jednocześnie
    addAbbreviations(abbreviations, context = 'general') {
        Object.entries(abbreviations).forEach(([abbr, expansion]) => {
            this.addAbbreviation(abbr, expansion, context);
        });
    }
    
    // Pobierz definicję w kontekście
    getExpansion(abbr, context = 'general') {
        const abbrMap = this.abbreviations.get(abbr);
        if (!abbrMap) return null;
        
        return abbrMap.get(context) || abbrMap.get('general') || null;
    }
    
    // Automatycznie dodaj elementy abbr w kontekście
    processSection(element, context = 'general') {
        const textNodes = this.getTextNodes(element);
        
        this.abbreviations.forEach((contextMap, abbr) => {
            const expansion = this.getExpansion(abbr, context);
            if (!expansion) return;
            
            const regex = new RegExp(`\\b${this.escapeRegex(abbr)}\\b`, 'g');
            
            textNodes.forEach(node => {
                if (regex.test(node.textContent) && !this.processed.has(node)) {
                    this.wrapAbbreviation(node, abbr, expansion);
                    this.processed.add(node);
                }
            });
        });
    }
    
    wrapAbbreviation(textNode, abbr, expansion) {
        const parent = textNode.parentNode;
        const text = textNode.textContent;
        const regex = new RegExp(`\\b(${this.escapeRegex(abbr)})\\b`, 'g');
        
        if (regex.test(text)) {
            const newHTML = text.replace(regex, 
                `<abbr title="${this.escapeHtml(expansion)}">$1</abbr>`
            );
            
            const wrapper = document.createElement('span');
            wrapper.innerHTML = newHTML;
            
            // Zastąp węzeł tekstowy nowym elementem
            parent.insertBefore(wrapper, textNode);
            parent.removeChild(textNode);
        }
    }
    
    getTextNodes(element) {
        const textNodes = [];
        const walker = document.createTreeWalker(
            element,
            NodeFilter.SHOW_TEXT,
            null,
            false
        );
        
        let node;
        while (node = walker.nextNode()) {
            if (node.textContent.trim()) {
                textNodes.push(node);
            }
        }
        
        return textNodes;
    }
    
    escapeRegex(string) {
        return string.replace(/[.*+?^${}()|[\]\\]/g, '\\$&');
    }
    
    escapeHtml(string) {
        return string.replace(/[&<>"']/g, (match) => {
            const escapeMap = { 
                '&': '&', '<': '<', '>': '>', 
                '"': '"', "'": ''' 
            };
            return escapeMap[match];
        });
    }
    
    // Generuj słowniczek dla danego kontekstu
    generateGlossary(context = 'general') {
        const glossary = [];
        
        this.abbreviations.forEach((contextMap, abbr) => {
            const expansion = contextMap.get(context) || contextMap.get('general');
            if (expansion) {
                glossary.push({ abbr, expansion });
            }
        });
        
        return glossary.sort((a, b) => a.abbr.localeCompare(b.abbr));
    }
}

System automatycznego słowniczka

            
class AutoGlossaryGenerator {
    constructor(options = {}) {
        this.minOccurrences = options.minOccurrences || 2;
        this.commonWords = new Set(options.commonWords || []);
        this.glossaryContainer = options.container || '#abbreviations-glossary';
    }
    
    // Znajdź wszystkie potencjalne skróty
    findAbbreviations() {
        const text = document.body.textContent;
        const abbreviations = new Map();
        
        // Wzorce dla różnych typów skrótów
        const patterns = [
            /\b[A-Z]{2,6}\b/g,  // Akronimy
            /\b[A-Z][a-z]*\.[A-Za-z.]*/g  // Skróty z kropkami
        ];
        
        patterns.forEach(pattern => {
            const matches = text.match(pattern) || [];
            matches.forEach(match => {
                if (!this.commonWords.has(match)) {
                    abbreviations.set(match, (abbreviations.get(match) || 0) + 1);
                }
            });
        });
        
        // Filtruj według minimalnej liczby wystąpień
        return new Map([...abbreviations].filter(([abbr, count]) => 
            count >= this.minOccurrences
        ));
    }
    
    // Próbuj znaleźć definicje w tekście
    findDefinitions(abbreviations) {
        const definitions = new Map();
        const bodyText = document.body.textContent;
        
        abbreviations.forEach((count, abbr) => {
            // Szukaj wzorców: "Full Form (ABBR)" lub "ABBR (Full Form)"
            const patterns = [
                new RegExp(`([^(]+)\\s*\\(\\s*${this.escapeRegex(abbr)}\\s*\\)`, 'gi'),
                new RegExp(`${this.escapeRegex(abbr)}\\s*\\(([^)]+)\\)`, 'gi')
            ];
            
            patterns.forEach(pattern => {
                const matches = bodyText.match(pattern);
                if (matches && matches.length > 0) {
                    const match = matches[0];
                    const fullForm = match.replace(abbr, '').replace(/[()]/g, '').trim();
                    if (fullForm && fullForm !== abbr) {
                        definitions.set(abbr, fullForm);
                    }
                }
            });
        });
        
        return definitions;
    }
    
    // Generuj HTML słowniczka
    generateGlossaryHTML(definitions) {
        if (definitions.size === 0) return '';
        
        const entries = Array.from(definitions.entries())
            .sort(([a], [b]) => a.localeCompare(b))
            .map(([abbr, definition]) => `
                <dt id="abbr-${abbr.toLowerCase()}"><abbr>${abbr}</abbr></dt>
                <dd>${definition}</dd>
            `).join('');
        
        return `
            <section id="auto-abbreviations-glossary">
                <h2>Słowniczek skrótów</h2>
                <dl>
                    ${entries}
                </dl>
            </section>
        `;
    }
    
    // Dodaj linki do słowniczka
    linkToGlossary(definitions) {
        definitions.forEach((definition, abbr) => {
            const regex = new RegExp(`\\b${this.escapeRegex(abbr)}\\b`, 'g');
            const elements = document.querySelectorAll('p, span, div');
            
            elements.forEach(element => {
                if (element.innerHTML.includes(abbr)) {
                    element.innerHTML = element.innerHTML.replace(regex, 
                        `<a href="#abbr-${abbr.toLowerCase()}">${abbr}</a>`
                    );
                }
            });
        });
    }
    
    // Główna funkcja
    generate() {
        const abbreviations = this.findAbbreviations();
        const definitions = this.findDefinitions(abbreviations);
        
        if (definitions.size > 0) {
            const glossaryHTML = this.generateGlossaryHTML(definitions);
            
            // Dodaj słowniczek na koniec dokumentu
            const container = document.querySelector(this.glossaryContainer) || 
                             document.querySelector('main') || 
                             document.body;
            
            const glossaryElement = document.createElement('div');
            glossaryElement.innerHTML = glossaryHTML;
            container.appendChild(glossaryElement);
            
            // Dodaj linki
            this.linkToGlossary(definitions);
        }
        
        return definitions;
    }
    
    escapeRegex(string) {
        return string.replace(/[.*+?^${}()|[\]\\]/g, '\\$&');
    }
}

// Użycie
const autoGlossary = new AutoGlossaryGenerator({
    minOccurrences: 2,
    commonWords: ['IT', 'OK', 'TV', 'DVD', 'USB'], // Wyklucz powszechne skróty
    container: 'main'
});

// Automatycznie generuj słowniczek
document.addEventListener('DOMContentLoaded', () => {
    const definitions = autoGlossary.generate();
    console.log('Wygenerowano definicje dla:', definitions);
});

        


        

            
Częste błędy


            

                
❌ Brak wyjaśnień dla skrótów

                
<!-- Nieprawidłowe -->
<p>API REST zwraca dane w formacie JSON.</p>
<p>CEO omówił strategię AI z zespołem HR.</p>

<!-- Prawidłowe -->
<p><abbr title="Application Programming Interface">API</abbr> 
<abbr title="Representational State Transfer">REST</abbr> 
zwraca dane w formacie <abbr title="JavaScript Object Notation">JSON</abbr>.</p>

            


            

                
❌ Używanie skrótów jako nazw własnych

                
<!-- Nieprawidłowe -->
<abbr title="International Business Machines">IBM</abbr> to znana firma.

<!-- Prawidłowe - IBM to nazwa firmy, nie skrót wymagający wyjaśnienia -->
<p>IBM to znana firma technologiczna.</p>

            


            

                
❌ Puste lub nieprecyzyjne atrybuty title

                
<!-- Nieprawidłowe -->
<abbr title="">API</abbr>
<abbr title="API">API</abbr>
<abbr title="To jest API">API</abbr>

<!-- Prawidłowe -->
<abbr title="Application Programming Interface">API</abbr>

            


            

                
❌ Nadużywanie elementu abbr

                
<!-- Nieprawidłowe - każde wystąpienie -->
<p><abbr title="HyperText Markup Language">HTML</abbr> to język znaczników. 
Każda strona używa <abbr title="HyperText Markup Language">HTML</abbr> 
do strukturyzacji treści <abbr title="HyperText Markup Language">HTML</abbr>.</p>

<!-- Prawidłowe - pierwsze wystąpienie lub wybrane -->
<p><abbr title="HyperText Markup Language">HTML</abbr> to język znaczników. 
Każda strona używa HTML do strukturyzacji treści HTML.</p>

            


            

                
❌ Niewłaściwe tłumaczenia skrótów

                
<!-- Nieprawidłowe -->
<abbr title="Częstowane Pytania">FAQ</abbr>

<!-- Prawidłowe -->
<abbr title="Frequently Asked Questions - Często Zadawane Pytania">FAQ</abbr>
<!-- lub -->
Frequently Asked Questions (<abbr>FAQ</abbr>) - Często Zadawane Pytania
