1.2.6 Język migowy (nagranie) (AAA)

Poziom AAA

Oficjalna dokumentacja W3C

Opis

Zapewnione jest tłumaczenie na język migowy dla wszystkich nagrań audio w multimediach zsynchronizowanych.

Intencja

Celem tego kryterium sukcesu jest zapewnienie osobom głuchym, dla których język migowy jest podstawowym sposobem komunikacji, dostępu do treści audio w nagraniach. Dla wielu osób głuchych od urodzenia lub wczesnego dzieciństwa, język migowy jest pierwszym i najbardziej naturalnym językiem, a język pisany może być trudniejszy do zrozumienia.

Podczas gdy napisy rozszerzone (kryterium 1.2.2) zapewniają transkrypcję dialogów, tłumaczenie na język migowy dodatkowo przekazuje intonację, emocje i inne aspekty komunikacji, które mogą być trudne do oddania w tekście.

Przykłady

Implementacja tłumaczenia na język migowy

Dobry przykład - wideo z tłumaczem języka migowego (PJM)

Film edukacyjny z tłumaczem języka migowego widocznym w prawym dolnym rogu:

[Tłumacz PJM] 
<video controls width="800">
    <source src="../videos/presentation-with-sign-language.mp4" type="video/mp4">
    <track kind="captions" src="../videos/captions.vtt" label="Polski" default>
    Twoja przeglądarka nie obsługuje znacznika wideo.
</video>

W tym przykładzie główny film zawiera już wbudowanego tłumacza języka migowego. Osoba migająca jest widoczna przez cały czas trwania materiału w prawym dolnym rogu ekranu, co umożliwia osobom głuchym pełne zrozumienie treści w ich naturalnym języku.

Dobry przykład - oddzielna ścieżka wideo z tłumaczem języka migowego

Implementacja przełączalnego tłumaczenia na język migowy:

[Tłumacz PJM] 
<div class="video-container">
    <video id="main-video" controls width="800">
        <source src="../videos/example-video.mp4" type="video/mp4">
        <track kind="captions" src="../videos/captions.vtt" label="Polski" default>
        Twoja przeglądarka nie obsługuje znacznika wideo.
    </video>
    
    <div id="sign-language-container" class="sign-language">
        <video id="sign-language-video" width="250" muted>
            <source src="../videos/sign-language.mp4" type="video/mp4">
        </video>
    </div>
</div>

<div class="controls">
    <button id="toggle-sign-language">Pokaż/ukryj język migowy</button>
</div>

<script>
    const mainVideo = document.getElementById('main-video');
    const signVideo = document.getElementById('sign-language-video');
    const toggleButton = document.getElementById('toggle-sign-language');
    const signContainer = document.getElementById('sign-language-container');
    
    // Synchronizacja odtwarzania
    mainVideo.addEventListener('play', () => {
        signVideo.currentTime = mainVideo.currentTime;
        signVideo.play();
    });
    
    mainVideo.addEventListener('pause', () => {
        signVideo.pause();
    });
    
    mainVideo.addEventListener('seeking', () => {
        signVideo.currentTime = mainVideo.currentTime;
    });
    
    // Przełączanie widoczności tłumacza
    toggleButton.addEventListener('click', () => {
        if (signContainer.style.display === 'none') {
            signContainer.style.display = 'block';
            toggleButton.setAttribute('aria-pressed', 'true');
        } else {
            signContainer.style.display = 'none';
            toggleButton.setAttribute('aria-pressed', 'false');
        }
    });
</script>

Ten przykład pokazuje implementację, w której tłumacz języka migowego jest wyświetlany w osobnym oknie, które można pokazać lub ukryć. Oba filmy (główny i z tłumaczem) są zsynchronizowane, dzięki czemu tłumaczenie dokładnie odpowiada dźwiękowi. To rozwiązanie daje użytkownikowi większą kontrolę nad sposobem wyświetlania tłumaczenia.

Problem: Ten film zawiera napisy rozszerzone (co spełnia kryterium 1.2.2 poziom A), ale nie zapewnia tłumaczenia na język migowy. Osoby głuche, dla których język migowy jest pierwszym językiem, będą miały ograniczone zrozumienie treści, szczególnie jeśli zawiera ona złożone pojęcia lub specjalistyczne terminy.

Najlepsze praktyki tworzenia treści z językiem migowym

  • Widoczność tłumacza - Tłumacz języka migowego powinien być dobrze widoczny, zazwyczaj w rogu ekranu, z kontrastowym tłem umożliwiającym dobrą widoczność rąk i mimiki.
  • Rozmiar okna - Okno z tłumaczem powinno być odpowiednio duże (zwykle co najmniej 1/6 wysokości ekranu), aby gesty i mimika były czytelne.
  • Oświetlenie - Tłumacz powinien być dobrze oświetlony, bez cieni na twarzy i dłoniach.
  • Właściwy dialekt - Należy używać odpowiedniego dialektu języka migowego dla danej lokalizacji (np. Polski Język Migowy - PJM).
  • Profesjonalny tłumacz - Zawartość powinna być tłumaczona przez profesjonalnego tłumacza języka migowego, posiadającego doświadczenie w tłumaczeniu specjalistycznych treści.
  • Synchronizacja - Tłumaczenie powinno być dobrze zsynchronizowane z treścią audio.

Wyzwania i ograniczenia

  • Różne języki migowe - Na świecie istnieje wiele różnych języków migowych, które nie są od siebie wzajemnie zrozumiałe (np. Polski Język Migowy, Amerykański Język Migowy, Brytyjski Język Migowy itp.).
  • Koszt - Zapewnienie profesjonalnego tłumaczenia na język migowy może być kosztowne, szczególnie dla dużych ilości treści.
  • Przeładowanie informacyjne - Jeśli wideo zawiera już wiele elementów wizualnych, dodanie tłumacza języka migowego może być trudne bez zakłócania głównej treści.
